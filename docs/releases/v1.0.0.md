## ries-rs 1.0.0

`ries-rs` 1.0.0 is the first stable release of the Rust reimplementation of
Robert P. Munafo's RIES inverse equation solver.

### Highlights

- Faithful core RIES search model with modern Rust implementation
- Deterministic mode for reproducible output ordering
- Structured JSON output and run manifests for automation and research workflows
- WebAssembly bindings for browser, bundler, and Node.js usage
- Python bindings via PyO3 and `maturin`
- PSLQ integer-relation mode in the CLI
- Public benchmark artifacts and parity tracking docs

### Included Artifacts

- Linux, macOS, and Windows CLI binaries
- WebAssembly package tarball with `pkg`, `pkg-node`, and `pkg-bundler`
- Python wheel artifacts built from `ries-py/`

### Notes

- For reproducible runs, use `--deterministic` together with `--json` and
  `--emit-manifest`.
- The browser UI can be hosted as a static bundle; see `web/README.md`.
- Citation metadata lives in `CITATION.cff`. Record a Zenodo DOI in project
  metadata only after archival has completed and the DOI has been verified.

### Compatibility

This release tracks behavior against both the original RIES and the
`clsn/ries` fork. Core search and classic-style workflows are implemented, but
exact ordering and complexity values may still differ on some targets because
the internal engine is Rust-native.

See `CHANGELOG.md` and `docs/PARITY_STATUS.md` for the detailed change and
compatibility history.
