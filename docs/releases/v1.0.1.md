## ries-rs 1.0.1

`ries-rs` 1.0.1 is a patch release focused on distribution and release
automation.

### Highlights

- Automated tag-driven publishing to crates.io and PyPI from the GitHub release
  workflow
- Added Python source distribution (`sdist`) artifacts alongside platform wheel
  builds
- Hardened release reruns by skipping registry uploads for versions or files
  that are already published

### Included Artifacts

- Linux, macOS, and Windows CLI binaries
- WebAssembly package tarball with `pkg`, `pkg-node`, and `pkg-bundler`
- Python wheel artifacts built from `ries-py/`
- Python source distribution (`ries_rs-*.tar.gz`)

### Notes

- The PyPI package now has package-local metadata and readme content so source
  distributions can be built cleanly with `maturin sdist`.
- Registry publishing is part of the release gate for this version: the GitHub
  release is created only after crates.io and PyPI publication succeed.

See `CHANGELOG.md` for the detailed patch history.
