## ries-rs 1.1.0

`ries-rs` 1.1.0 is a polish release focused on release integrity, public
documentation, and the canonical web experience around the WASM build.

### Highlights

- Added automated release-integrity checks for version sync, canonical
  homepage metadata, release notes presence, and README assets
- Promoted `https://maxwellsantoro.com/projects/ries-rs` as the canonical
  public landing page for packages and citation metadata
- Clarified the handoff between release artifacts in `ries-rs` and deployed
  demo validation in the website repository

### Included Artifacts

- Linux, macOS, and Windows CLI binaries
- WebAssembly package tarball with `pkg`, `pkg-node`, and `pkg-bundler`
- Python wheel artifacts built from `ries-py/`
- Python source distribution (`ries_rs-*.tar.gz`)

### Notes

- The canonical landing page explains the project and links directly to the
  live browser demo.
- This release keeps the browser build aligned with the same core solver shipped
  in the CLI, Rust crate, and Python bindings.
- GitHub releases may also be archived on Zenodo. Record a DOI in project
  metadata only after archival has completed and the DOI has been verified.

See `CHANGELOG.md` for the detailed release history.
